Self-attention, in one picture

For each token, attention computes weights over every other token and combines their values. Below: how the word "it" attends to the rest of the sentence — the link to "animal" stays bright while irrelevant ones fade.

Scaled dot-product attention

$$\mathrm{Attention}(Q,K,V) = \mathrm{softmax}\!\left(\tfrac{Q K^\top}{\sqrt{d_k}}\right) V$$
Each query $Q$ scores against all keys $K$, the scores become weights via softmax, and a weighted sum of values $V$ is returned.
Multi-head
$$\mathrm{MultiHead}(Q,K,V) = \mathrm{Concat}(h_1,\dots,h_8)\,W^O$$
Eight parallel attention "heads" let the model focus on different relationships at once.

Why it changed everything

  • Parallelism
    Unlike RNNs, every position is processed simultaneously — perfect for GPUs.
  • Constant path length
    Information between any two positions travels through one attention layer — long-range deps become easy.
  • Scales gracefully
    Performance kept improving with more parameters and data — the foundation of every modern foundation model.
