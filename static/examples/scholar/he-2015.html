The residual block

Instead of learning a direct mapping $H(x)$, learn the residual $F(x)=H(x)-x$ and add the input back via a shortcut.

The reformulation

Plain network
$$H(x)$$
"Learn the desired mapping directly" — gradients vanish for very deep nets.
Residual network
$$y = F(x, \{W_i\}) + x$$
Easier to push $F\to 0$ (identity) than to learn $H$ from scratch — gradients flow through the shortcut.
