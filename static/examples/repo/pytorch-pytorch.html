What it does

Distilled from the README

PyTorch gives you a numpy-style array library where every operation is differentiable. You write a neural network as a regular Python class; backpropagation, GPU placement and distributed training come for free.

Originally built at Facebook AI Research, it's now the framework behind almost every major published ML paper, every modern open LLM, and a growing share of production inference too — especially since the `torch.compile` JIT in PyTorch 2 closed most of the perf gap with C++ frameworks.

  • ▸Autograd over arbitrary Python
  • ▸Native distributed training (FSDP, DDP)
  • ▸torch.compile for graph-mode speedups
  • ▸Strong CUDA, MPS and ROCm backends
  • ▸Massive ecosystem (TorchVision, TorchText, Lightning, …)
